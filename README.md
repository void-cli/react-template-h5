## npm scripts

```bash
npm run api-gen # 生成 api 模块代码
npm run cimg # 压缩 `src/assets/` 的图片
npm start # 本地开发
npm run build # 本地构建
npm run up-big # 上传大文件到 cdn
npm run dep-test # 部署 test 环境，配置平台使用 test 环境
npm run dep-temp # 部署 temp 环境，配置平台使用 dev 环境
npm run dep-prod # 部署 prod 环境，上 CDN，关闭 vconsole，配置平台使用 prod 环境
```
